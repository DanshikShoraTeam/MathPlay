MathPlay — Премиум математикалық оқыту платформасы (Educaplay стилінде)

Қалай іске қосу керек:

1. Виртуалды ортаны белсендіру:
   source venv/bin/activate

2. Тәуелділіктерді орнату:
   pip install -r requirements.txt

3. Деректер базасын дайындау және үлгілік ойындарды жүктеу:
   python manage.py migrate
   python populate_db.py

4. Серверді іске қосу:
   python manage.py runserver

5. Браузерде ашу:
   http://127.0.0.1:8000/
